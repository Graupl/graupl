# Form

Default Values

```jsx
// Form selectors.
$input-selector: "input" !default;
$textarea-selector: "textarea" !default;
$select-selector: "select" !default;
$label-selector: "label" !default;
$fieldset-selector: "fieldset" !default;

// Fieldset properties.
$fieldset-direction: column !default;
```
