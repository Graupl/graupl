Hello Graupl!

Graupl is a modular CSS framework with the goal of making it easy to theme a site.

This source code can be found in GitHub.

Graupl supports light and dark modes out-of-the-box.

Buttons

There are 4 colours of buttons supported by default, with plans to have a "graupl-theme" plugin that adds more.

Inverse Cards

Cards using the .inverse class will have the image placed on the right if it is a horizontal card and on the bottom if it is a vertical card.

This is mainly to provide a shortcut for horiztonal cards that you'd like to use the .bottom .right classes on when the card is forced to be vertical.
