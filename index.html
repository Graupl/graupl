Hello Graupl!

Graupl is a modular CSS framework with the goal of making it easy to theme a site.

This source code can be found in GitHub.

Graupl supports light and dark modes out-of-the-box.

Buttons

There are 4 colours of buttons supported by default, with plans to have a "graupl-theme" plugin that adds more.

Form elements

Graupl provides default styling for all form elements.
